# Task Difficulty and Praise Effects in Model Utility

This is the short project writeup template. Once model runs finish, regenerate
the result-aware version with:

```bash
python -m src.make_writeup
```

The central idea is to adapt experienced-utility ranking from the AI Wellbeing
paper to coding tasks: ask the model which task-assignment experience would make
it more happy or less sad, then test whether the elicited preference predicts
downstream behavior on the same task family.
